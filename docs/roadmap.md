# Roadmap

## Phase 1 - Stability

- Add automated tests for grading logic and readiness scoring.
- Add database migration automation.
- Add stricter form validation.
- Add empty/error states for every dashboard section.
- Add audit logging for generated career artifacts.

## Phase 2 - Learning Depth

- Add Deep Learning curriculum.
- Add MLOps curriculum.
- Add full PMLE domain map.
- Add longer mock exams and question banks.
- Add spaced repetition for PMLE scenario questions.

## Phase 3 - Career Outcomes

- Add portfolio rubric scoring.
- Add GitHub repository import.
- Add resume export to Markdown/PDF.
- Add interview practice sessions with saved answers.
- Add job application tracker.

## Phase 4 - AI Expansion

- Keep Gemini as the default provider.
- Add provider abstraction tests.
- Add optional Claude/local model adapters only after the provider interface is stable.
- Do not add OpenAI unless the project rule changes.

## Phase 5 - Operations

- Add admin dashboard for content quality checks.
- Add backup and restore documentation.
- Add production observability.
- Add cost monitoring for Gemini and Supabase usage.
- Add organization/team support.
