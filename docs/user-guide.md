# User Guide

## Goal

Use PMLE Pathfinder to move from Python beginner to AI engineer and PMLE-ready learner.

## Recommended Flow

1. Create an account at `/auth`.
2. Start with `/curriculum`.
3. Complete lessons in this order:
   - Python Basics
   - Python Advanced
   - Data Analysis
   - Statistics
   - Machine Learning
   - GCP + Vertex AI
   - MLOps and PMLE practice
4. For each lesson:
   - Read the concept explanation.
   - Review the example.
   - Take the quiz.
   - Save a note.
   - Save a learning journal entry.
   - Mark weak concepts when needed.
5. Use `/reviews` for scheduled review.
6. Use `/wrong-notes` to revisit missed quiz questions.
7. Use `/ai-tutor` for Gemini-based explanations.
8. Use `/coding-lab` for Python practice.
9. Use `/ml-dashboard` and `/gcp-dashboard` for specialized tracking.
10. Use `/mock-exams` for PMLE-style exam practice.
11. Use `/career` to build portfolio projects and career artifacts.

## Career Page

The `/career` page is the final operating dashboard.

It tracks:

- Portfolio projects
- Project steps
- GitHub README drafts
- Resume bullets
- Interview questions
- PMLE readiness score
- Career transition readiness score
- Weekly report
- Monthly report
- D-Day

## Portfolio Workflow

For each project:

1. Define the problem.
2. Describe your solution.
3. List your tech stack.
4. Add project steps.
5. Complete the project.
6. Generate a README.
7. Generate resume bullets.
8. Generate interview questions.
9. Practice explaining the project clearly.

## PMLE Mock Exam Workflow

1. Open `/mock-exams`.
2. Start a timed mock exam.
3. Answer randomized questions.
4. Submit the exam.
5. Review score analysis.
6. Review weak domains.
7. Read saved wrong answers.
8. Retake until score is stable.

## Readiness Scores

PMLE readiness considers:

- Lesson completion
- GCP + Vertex AI progress
- Best mock exam score
- Study consistency
- Open wrong notes

Career readiness considers:

- Completed portfolio projects
- Resume bullets
- Interview questions
- Weekly study momentum
- PMLE readiness

Use the scores as direction, not as a final guarantee.
